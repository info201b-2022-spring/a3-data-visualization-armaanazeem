The visualisation below is interactive: hover your cursor over the graph to see more concise data.

The graph to the right illustrates the chi-squared value of incarcerated ethnnicity groups among the United States.

The chi-squared test depicted in this graph takes each incarcerated ethnicity group (Asian-American/Pacific Islander, Black, Latinx, White, and Other) and their respective proportion to the total incarcerated population of each state, and calculates a chi-squared value for each set of proportions. The lower the chi-squared value, the more "uniform" and evenly represented the different incarcerated ethnicity groups are for each state.

Some bias present within this visualisation is that Alaska and Hawaii are omitted. Although Alaska is an integral factor within the scope of this study, the R package utilised to confiure the geographical map did not harbour data for Alaska to be represented. Furthermore, the states of Connecticut, Deleware, Rhode Island, and Vermont are excluded (see the Summary page for more information).
